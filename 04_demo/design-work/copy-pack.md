# Copy pack

## Voice and tone guardrails
- clinician-facing
- grounded
- concise
- calm and serious
- assistive, not autonomous
- no agent / prompt / model jargon
- no consumer-genetics language
- no overclaiming diagnosis, eligibility, or certainty

## Core module 1
- Heading: Recommendation
- Body: Supports specialist review in the selected referral lane
- Support copy: Grounded in the uploaded report, extracted fields, evidence sources, and narrow decision logic. Final action remains clinician-reviewed.

## Core module 2
- Heading: Evidence bundle
- Body: Variant interpretation, phenotype context, and pathway evidence appear together in one reviewable bundle.
- Support copy: Each evidence card should show source context and missing-data state, not just a verdict.

## Core module 3
- Heading: Reviewer actions
- Body: Confirm key fields, add a note, then approve and export or hold for more data.
- Support copy: The reviewer decision is the visible endpoint of the page.

## Copy constraints
- Avoid these terms:
  - AI copilot
  - system prompt
  - model reasoning
  - confidence score >85
  - autonomous eligibility
  - diagnosis
  - instant answer
  - generic genomic report
- Prefer these terms:
  - recommendation
  - evidence
  - review
  - missing data
  - uncertainty
  - approve and export
  - hold for more data
  - source report
  - extracted fields
