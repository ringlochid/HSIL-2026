# Problem Scope

## Core problem (initial)

Clinicians spend time reconciling fragmented case input (symptoms, genomic findings, imaging summaries) into a structured clinical next-step recommendation.

The project aims to reduce friction at a narrow decision point with a reviewable, explainable draft output.

## User persona

- Clinical users (primarily ophthalmology / inherited retinal disease pathway)
- Need quick, structured interpretation support before deciding next action

## Scope boundaries

- In-scope: structured summary + interpretation draft + traceability
- Out-of-scope (MVP): real-time autonomous diagnosis, treatment prescribing, broad cross-specialty claims

## Success criteria for research phase

1. Clinical workflow is credible and specific.
2. Data assumptions are explicit and manageable.
3. Outputs are clearly marked as assistive and human-reviewed.
